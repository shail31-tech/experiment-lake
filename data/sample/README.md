# Sample Data

This directory stores local sample data generated in Phase 1.

Generated files:

- `users.csv`: one row per simulated user
- `experiment_assignments.csv`: one row per user per experiment assignment
- `events.csv`: event-level data for easy local inspection
- `events.jsonl`: newline-delimited JSON for later S3 ingestion

These files are intentionally ignored by git because they are generated artifacts.
